This page is served over HTTPS from GitHub Pages, but the car's API is plain HTTP on a private IP (e.g. 192.168.4.1). Most browsers block that request from a public HTTPS page (mixed content / private-network restrictions). If "Connect" can't reach the car, open this same docs/index.html from a local file or local HTTP server instead — see the project README.
